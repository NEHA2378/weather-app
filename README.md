# 🌤️ Weather App

A real-time weather app built with HTML, CSS, and JavaScript using the OpenWeatherMap API.

## Features
- Search weather by city name
- Shows temperature, humidity, wind speed, pressure, and feels-like
- Responsive design

## Tech Stack
HTML · CSS · JavaScript · OpenWeatherMap REST API
